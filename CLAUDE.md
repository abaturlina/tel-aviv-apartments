# Tel Aviv Apartment Finder 🏠

## Цель проекта
Автоматизированный поиск квартир в Тель-Авиве для аренды.
Парсим объявления, проверяем критерии, отображаем на интерактивной карте.

---

## Критерии поиска

| Параметр   | Значение                    | Обязательно? |
|------------|-----------------------------|--------------|
| Комнаты    | 3                           | ✅ Да         |
| Mamad      | есть                        | ✅ Да         |
| Parking    | есть                        | ⚡ Желательно |
| Цена       | до 10,000 ILS/мес           | ✅ Да         |
| Цена макс  | до 12,000 ILS/мес           | ⚡ С пометкой |
| Город      | Тель-Авив и ближние районы  | ✅ Да         |

Квартира **проходит фильтр**, если:
- rooms == 3
- mamad == true
- price <= 10000 → статус "good"
- price 10001–12000 → статус "over_budget" (показывать, но пометить)
- price > 12000 → **не сохранять**

---

## Структура данных (apartments.json)

Каждая квартира — объект со следующими полями:

```json
{
  "id": "уникальный хэш от URL",
  "url": "https://...",
  "source": "yad2 / homeless / madlan / manual",
  "address": "полный адрес на иврите или английском",
  "lat": null,
  "lng": null,
  "rooms": 3,
  "price": 9500,
  "mamad": true,
  "parking": false,
  "floor": null,
  "size_sqm": null,
  "notes": "любые дополнительные детали из объявления",
  "status": "new",
  "added_at": "2026-05-13T10:00:00"
}
```

Поле `status` принимает значения:
- `"new"` — только добавлена
- `"good"` — цена до 10,000
- `"over_budget"` — цена 10,001–12,000
- `"interested"` — я отметил вручную
- `"rejected"` — я отверг вручную
- `"contacted"` — я уже связался

## Технологии

- **Python 3** для парсинга и обработки
- **requests + BeautifulSoup4** для чтения HTML страниц
- **Nominatim (OpenStreetMap)** для геокодирования — БЕСПЛАТНО
- **Leaflet.js** для карты — БЕСПЛАТНО, без API-ключа
- **JSON** для хранения данных

## Важные сайты с объявлениями

- `yad2.co.il` — основной израильский портал
- `homeless.co.il` — популярный сайт аренды
- `madlan.co.il` — агрегатор
- `facebook.com/groups` — группы в Facebook (парсинг сложнее)

## Как использовать скрипт

```bash
# Установить зависимости (один раз)
pip install -r requirements.txt

# Добавить квартиру по URL
python scraper.py "https://www.yad2.co.il/item/xxxxx"

# Добавить несколько URL из файла
python scraper.py --file urls.txt

# Обновить координаты для квартир без lat/lng
python geocode.py

# Открыть карту
open map.html   # macOS
start map.html  # Windows
```

## Правила для Claude Code

1. **Всегда читай этот файл** перед началом работы
2. **Не удаляй данные** из apartments.json — только добавляй или обновляй
3. **Проверяй дубликаты** по полю `url` перед добавлением
4. **Пиши комментарии** в коде на русском или английском
5. **После изменений** в scraper.py — обязательно тестируй на одном URL
6. **Обновляй decisions.md** если что-то сломалось или изменилось на сайте
